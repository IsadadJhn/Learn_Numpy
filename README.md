# NumPy Fundamentals

Repository ini berisi kumpulan latihan dan contoh kode NumPy yang saya pelajari, mulai dari konsep dasar hingga operasi umum.

## Materi yang dipelajari
- Pembuatan array (1D, 2D, nD)
- Shape,dimensi array,reshape
- Indexing dan slicing
- Broadcasting
- Operasi matematika vektorisasi
- Aggregate functions (sum, mean, min, max, std, var)
- Random number generation
- Save & load array (.npy dan .npz)

## Tujuan
Repository ini dibuat sebagai dokumentasi pembelajaran dan referensi pribadi dalam mempelajari NumPy.
